# Frontend Code Repository

Please drop your AI-generated frontend UI code inside this `frontend` folder.

If it is a React/Vue/Svelte app, you can just unpack the entire project structure right here. If it is standard HTML/CSS/JS, place the `index.html` and assets in this folder.

The backend API is running on FastAPI and can be accessed at `http://127.0.0.1:8000`. The primary endpoint for chatting is `POST /chat`.
